# FacilLibrary: An RPG library made with LÖVE
### By Julio :)

## What is this?  

FacilLibrary (FL) is a set of libraries which make creating RPG's much easier in the Lua's framework, [LÖVE](https://github.com/love2d/love). These are free to use, licensed under the MIT license, and built in a modular way so you can use these anywhere.

## Whats included?

LRPGC includes many utilities to accelerate your workflow, such as:

- A basic timer
- RPG-style textbox with typewriter effect
- Utilities such as a fade system
- Sample for main character and a party
- Inventory system
- Sample for enemies
- Sample for RPG-style turn based combat
